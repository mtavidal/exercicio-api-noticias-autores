# Repositório - API de Notícias e Autores
#### Repositório de uma API para cadastro de notícias e seus autores. Foram realizadas autenticações com token e validações das informções recebidas no body.

#### Foi utilizado:
- Arquitetura MVC
- NodeJS
- dotenv
- express
- mysql2
- sequelize
- bcryptjs
- express-validator
- jsonwebtoken
